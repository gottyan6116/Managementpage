# 02 — デザインシステム / UI 完全再現仕様

> スクリーンショットから抽出した視覚仕様。色・余白・角丸・影は**必ずこの値**を使う。
> 数値は厳密な実測ではなく「再現に十分な近似」。実装後にスクショと比較して微調整する。

## 1. デザインの第一印象（再現の肝）

- **明るく・やわらかい・上質**。背景はほぼ白、カードは白＋ごく淡い影。
- **左サイドバーだけが主役カラー**: 青グラデーション＋山と湖の写真をうっすら合成し、
  ガラス（frosted glass）質感。メニューは半透明の白でアクティブ表現。
- 角丸は大きめ（カード 16px、ボタン 10〜12px）。影は淡く広い。
- アクセントは **青** を中心に、案件種別で **ティール / パープル / オレンジ** を併用。

- フォント: **Inter**（英数）＋ **Noto Sans JP**（和文）。`font-feature-settings:"palt"` で和文の詰め。

## 4. スペーシング・角丸・影

```css
:root {
  /* spacing scale (px) */ --sp-1:4; --sp-2:8; --sp-3:12; --sp-4:16; --sp-5:20; --sp-6:24; --sp-8:32;
  /* radius */ --r-sm:8px; --r-md:10px; --r-lg:12px; --r-xl:16px; --r-2xl:20px; --r-full:9999px;
  /* shadow（淡く広い） */
  --shadow-card:  0 1px 2px rgba(16,24,40,.04), 0 4px 16px rgba(16,24,40,.06);
  --shadow-pop:   0 8px 30px rgba(16,24,40,.12);
  --shadow-side:  0 8px 40px rgba(40,80,160,.25); /* サイドバーの浮き */
}
```
- カード: 背景 `--surface` / 角丸 `--r-xl` / 影 `--shadow-card` / 内側 padding `--sp-6`。
- カード間ギャップ: `--sp-5`（20px）。
- ページ外周 padding: 32px。

## 5. レイアウトグリッド

```
┌── Sidebar 248px ──┬──────────── Main（残り全幅）────────────┐
│ (固定/折りたたみ) │  Header 64px                            │
│                   ├─────────────────────────────────────────┤
│                   │  Content（padding 32px, max-width 1600）│
└───────────────────┴─────────────────────────────────────────┘
```
- サイドバー幅: 展開 248px / 折りたたみ 80px（アイコンのみ）。
- ヘッダー高さ: 64px、白背景、下境界 `--border`。
- メインは内側に角丸 `--r-2xl` の白い大パネルで囲む（スクショの「白い大きな角丸の中に全部入っている」表現）。

## 6. 主要コンポーネント仕様

### KPI カード
- 左上に丸アイコンチップ（48px 円、淡い色背景＋濃い色アイコン）。
- 見出し（14px secondary）→ 大数値（32px bold）→ 補助（前月比 +2↑ など、緑/赤）。
- 右側にミニグラフ（スパークライン折れ線 / ドーナツ / 棒グラフ）。Recharts。

### バッジ（Badge）
- 角丸 `--r-full`、padding 2px 10px、12px/600。色は §2 の表に従う。

### アバター（Avatar / AvatarGroup）
- 円形 28px、白 2px のリング。重なり表示は -8px マージン。超過は「+N」チップ。
- 各メンバーの `color` を枠やフォールバックに使用。

### 進捗バー（ProgressBar）
- 高さ 6px、角丸 full、トラック `--surface-muted`、フィルはプロジェクト色。
- 右に `72%` などの数値（12px / 600）。

### テーブル
- ヘッダー: 文字 `--text-muted` 12px、背景なし（or 極淡）、下境界。
- 行高さ 56px、行 hover で `--surface-muted`。境界は行下のみ淡線。
- 左端にラジオ/チェックの円。

### タブ
- アクティブ: 文字 `--brand-600` ＋ 下線 2px `--brand-600`。非アクティブ: `--text-secondary`。
- タブ横に件数（例「すべて 120」）を淡色で。

### ドーナツチャート（ステータス分布）
- 中心に合計（「12 件」）。凡例は右に色ドット＋ラベル＋件数＋(％)。

## 7. ガント専用の視覚仕様（再現の最難所）

- タイムライン列: 週単位ヘッダ（例「5/5 - 5/11」）＋下に曜日（月火水木金土日）。
- 1 日の列幅: 約 28〜32px（ズームで可変）。
- バー: 高さ 16px、角丸 full、プロジェクト色（親は濃い帯、子タスクは淡め）。
  - 進捗はバー内の塗り分け or 右端に `%`。
  - バー右に終了日ラベル（例「5/23」）。
- マイルストーン: ◆（菱形）＋日付ラベル。
- 依存関係: 前タスク終端 → 次タスク始端へ L 字のコネクタ線（`--text-muted`）。
- 今日ライン: 縦の青ライン＋上部に「今日 5/16」バッジ（`--brand-600`）。
- 親プロジェクト行は背景わずかに濃く、▼ で開閉。

## 8. アニメーション / マイクロインタラクション（UX）

- hover/フォーカス: 150ms ease。カード hover でわずかに影を強調。
- サイドバー項目: アクティブ化に 200ms のスライド/フェード。
- 数値・グラフ: 初回マウントで 600ms のカウントアップ / 描画アニメ（控えめ）。
- D&D: ドラッグ中カードは影強調＋少し傾ける（2deg）。
- トースト: 右下、楽観的更新の確定/失敗を通知。
- `prefers-reduced-motion` を尊重し、無効時はアニメ最小化。
