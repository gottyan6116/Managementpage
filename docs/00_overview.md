# 00 — 全体像と進め方

## このプロジェクトのゴール

提供された 3 つのスクリーンショット（Todo ダッシュボード / ガントチャート / 担当案件）を
**UI 完全再現**しつつ、Supabase をバックエンドに **実際に動く** マネジメントアプリ ProManage を作る。

- フェーズ 1: フロントの完全再現（ダミーデータで全画面が動く）
- フェーズ 2: Supabase 接続（認証・CRUD・永続化）
- フェーズ 3: 仕上げ（リアルタイム・通知・アクセシビリティ・レスポンシブ）

## 進め方の原則

1. **小さく刻む**: `docs/06_roadmap.md` のタスク順に。1 タスク＝1 PR 相当。
2. **再現が正義**: 各画面はスクショと並べて差分ゼロを目指す。
3. **ダミー → 本物**: まずダミーデータで見た目を完成させ、後で Supabase に差し替え。
   そのため **データ取得は最初からフック（`useTasks()` 等）に隠蔽**しておく。
4. **共通から個別へ**: デザイントークン → 共通コンポーネント → レイアウト → 各画面。

## マイルストーン

- **M1**: AppShell（サイドバー＋ヘッダー）＋ Todo 画面がダミーで完成
- **M2**: ガント・担当案件が完成（フロント 3 画面完成）
- **M3**: 残り 4 画面の枠 + Supabase 接続で Todo が本物データで動く
- **M4**: 全画面 Supabase 化・リアルタイム・通知・アクセシビリティ仕上げ
